# Dynamic structures in C

A collection of libraries for managing dynamic structures (stack, queue and list) in C.

## Usage

Basic stack:
```c
#include "stack.h"
```

Stack with linked nodes:
```c
#include "linked_node_stack.h"
```

Basic queue:
```c
#include "stack.h"
```

List:
```c
#include "stack.h"
```
